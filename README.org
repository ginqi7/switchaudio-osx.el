* Overview

A minimal Emacs package for switching audio output devices on macOS. It provides a simple interface to interact with [[https://github.com/deweller/switchaudio-osx][switchaudio-osx]] command-line tool.

* Prerequisites

- macOS
- [[https://github.com/deweller/switchaudio-osx][switchaudio-osx]] - Install via Homebrew:
  #+BEGIN_SRC sh
    brew install switchaudio-osx
  #+END_SRC

* Installation

** From Git

1. Clone the repository to your Emacs load path:
   #+BEGIN_SRC sh
     git clone https://github.com/ginqi7/switchaudio-osx.el ~/.emacs.d/lisp/switchaudio-osx
   #+END_SRC

2. Add the path to your =load-path= in Emacs config:
   #+BEGIN_SRC emacs-lisp
     (add-to-list 'load-path "~/.emacs.d/lisp/switchaudio-osx")
     (require 'switchaudio-osx)
   #+END_SRC

* Usage

** Commands

| Command                     | Description                                    |
|-----------------------------+------------------------------------------------|
| =M-x switchaudio-osx-list=   | List all available audio devices               |
| =M-x switchaudio-osx-switch= | Interactively switch to a different audio device |
| =M-x switchaudio-osx-mute=   | Mute/unmute audio (TODO)                       |

** Example Configuration

#+BEGIN_SRC emacs-lisp
  ;; Load the package
  (require 'switchaudio-osx)

  ;; Bind switch command to a key
  (global-set-key (kbd "C-c a s") 'switchaudio-osx-switch)

  ;; List devices
  (global-set-key (kbd "C-c a l") 'switchaudio-osx-list)
#+END_SRC

* API Reference

| Function                    | Description                          |
|-----------------------------+--------------------------------------|
| =switchaudio-osx-all=        | Get all audio devices as a list      |
| =switchaudio-osx-current=    | Get the current audio device         |
| =switchaudio-osx-active=     | Activate a device by device info     |
| =switchaudio-osx-list=       | Interactive: list all devices        |
| =switchaudio-osx-switch=     | Interactive: switch audio device     |
| =switchaudio-osx-mute=       | Interactive: mute/unmute (TODO)      |

* Customization

| Variable                     | Default Value              | Description                     |
|------------------------------+----------------------------+---------------------------------|
| =switchaudio-osx-command=     | =(executable-find "SwitchAudioSource")= | Path to the SwitchAudioSource command |

* Examples

** Get All Devices
#+BEGIN_SRC emacs-lisp
  (switchaudio-osx-all)
  ;; => List of device hash maps
#+END_SRC

** Get Current Device
#+BEGIN_SRC emacs-lisp
  (switchaudio-osx-current)
  ;; => Hash map of current device
#+END_SRC

** Switch to a Specific Device
#+BEGIN_SRC emacs-lisp
  (let ((device (car (switchaudio-osx-all))))
    (switchaudio-osx-active device))
#+END_SRC
