* Prerequisites

** Install
+ clone the project in your own emacs configuration files directory, like:
  #+BEGIN_SRC sh
    git clone https://github.com/ginqi7/switchaudio-osx.el ~/.emacs.d/lisp/switchaudio-osx;
  #+END_SRC

+ add the path to 'load-path'
  #+BEGIN_SRC emacs-lisp
    (add-to-list 'load-path "~/.emacs.d/lisp/switchaudio-osx")
  #+END_SRC

* Configuration
Here are a few variables that can be used for configuration:
#+BEGIN_SRC emacs-lisp
  (require 'switchaudio-osx)
